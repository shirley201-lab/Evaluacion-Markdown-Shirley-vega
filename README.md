# Evaluación de Conocimientos

## ¿Qué es un archivo Markdown y para qué sirve?
Un archivo Markdown es un texto plano con formato simple que permite crear documentos estructurados de forma clara y fácil de leer, especialmente en plataformas como GitHub.

## ¿Cuál es el propósito del archivo README.md en un repositorio?
El archivo README.md sirve para describir el proyecto, explicar su uso y proporcionar información importante a los usuarios.
